# Sales Performance Analysis Dashboard

## Overview
This project analyzes a sales dataset to identify key business insights such as top-performing products, regional performance, and sales trends.

## Objectives
- Understand sales performance across regions and categories
- Identify high-performing products
- Analyze monthly and seasonal trends
- Support business decision-making using data insights

## Tools Used
- Microsoft Excel (Data Cleaning, Pivot Tables)
- Power BI (Dashboard Visualization)

## Key Insights
- Identified top-performing products contributing to maximum revenue
- Analyzed regional sales trends to highlight high-performing areas
- Observed seasonal patterns in sales performance
- Highlighted opportunities for revenue improvement

## Dashboard Features
- Sales by Region
- Top Products Analysis
- Monthly Sales Trends
- Profit vs Revenue Comparison

## Outcome
This project demonstrates how data analysis and visualization can support business decision-making and improve overall performance.

## Project Structure
- `sales_data.csv` → dataset used for analysis
- `dashboard screenshots` → visual outputs
